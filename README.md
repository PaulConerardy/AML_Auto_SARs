# AML SAR Assistant

Assistant de rédaction de rapports d'activité suspecte (SAR) utilisant une architecture RAG (Retrieval-Augmented Generation).

## Objectifs
- Réduction du temps de rédaction des rapports SAR.
- Garantie de cohérence narrative (Qui, Quoi, Quand, Où, Pourquoi).
- Anonymisation PII automatique.
- Validation par agent ("Agent-as-a-Judge").

## Stack
- Frameworks : LangChain, LlamaIndex
- LLM : GPT-4o / Claude 3.5 Sonnet
- Vector Store : ChromaDB
- Interface : Streamlit
